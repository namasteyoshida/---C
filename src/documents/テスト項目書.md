# テスト項目書

| 機能 | テスト内容 | 期待結果 |
|------|-----------|----------|
| 一覧表示 | 初期表示時に経費一覧を表示する | 初期データ5件以上が一覧表示される |
| 一覧表示 | 経費が0件の場合 | 「登録された経費はありません。」と表示される |
| 承認状態表示 | 承認済みデータを表示する | 「✔承認済み」が表示される |
| 承認状態表示 | 未承認データを表示する | 「承認する」ボタンが表示される |
| 高額経費表示 | 金額が10,000円以上の経費を表示する | 金額が赤字で表示され、「（高額経費）」が表示される |
| カテゴリ絞り込み | 「すべて」を選択する | 全件表示される |
| カテゴリ絞り込み | 「交通費」を選択する | 交通費のみ表示される |
| カテゴリ絞り込み | 「消耗品」を選択する | 消耗品のみ表示される |
| カテゴリ絞り込み | 「会議費」を選択する | 会議費のみ表示される |
| カテゴリ絞り込み | 「その他」を選択する | その他のみ表示される |
| 金額並び替え | 「金額昇順」を選択する | 金額が小さい順に表示される |
| 金額並び替え | 「金額降順」を選択する | 金額が大きい順に表示される |
| 集計表示 | 初期表示時の集計を確認する | 件数・合計金額・未承認金額が正しく表示される |
| 集計表示 | 承認を実行する | 未承認金額が更新される |
| 集計表示 | 新規登録する | 件数・合計金額・未承認金額が更新される |
| 新規登録 | 必須項目を入力して登録する | 経費が一覧に追加され、未承認で登録される |
| 新規登録 | 登録後のフォームを確認する | 入力フォームが初期状態にリセットされる |
| 件名入力 | 件名を空欄で登録する | 「件名は必須入力です。」と表示される |
| 件名入力 | 31文字以上入力する | 「件名は30文字以内で入力してください。」と表示される |
| 件名入力 | 前後に空白を付けて入力する | 前後の空白が除去されて登録される |
| カテゴリ入力 | カテゴリ未選択で登録する | 「カテゴリは必須入力です。」と表示される |
| 金額入力 | 金額未入力で登録する | 「金額は必須入力です。」と表示される |
| 金額入力 | 0円を入力する | 「金額は1円以上で入力してください。」と表示される |
| 金額入力 | 300,001円を入力する | 「金額は300,000円以下で入力してください。」と表示される |
| 金額入力 | 150円を入力する | 「金額は100円単位で入力してください。」と表示される |
| 日付入力 | 日付未入力で登録する | 「日付は必須入力です。」と表示される |
| 用途メモ入力 | 101文字入力する | 「用途メモは100文字以内で入力してください。」と表示される |
| 承認機能 | 「承認する」ボタンをクリックする | 承認済みに更新され、「✔承認済み」が表示される |
